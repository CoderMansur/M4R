The fact that some nodes are not connected needs to be changed. We need to append the subgraphs and run feature extraction on each subgraph. Add features that relate to the extra subgraphs. or features indicatnig there are subgraphs.

pre_compute()[source]¶

This function pre-computes a set of calculations that are often reused by various functions. This saves time by not pre-computing the same data for multiple functions.
