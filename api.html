The centrality degree module¶

class hcga.Operations.centrality_degree.DegreeCentrality(G)[source]¶

Degree centrality class

feature_extraction(args)[source]¶

Compute the degree centrality for nodes.

The degree centrality for a node v is the fraction of nodes it is connected to.

G : graph
A networkx graph
args :
arg[0] Number of bins for calculating pdf of chosen distribution for SSE calculation
feature_list :list
List of features related to degree centrality.
Degree centrality calculations using networkx:
https://networkx.github.io/documentation/stable/_modules/networkx/algorithms/centrality/degree_alg.html#degree_centrality

The degree centrality values are normalized by dividing by the maximum possible degree in a simple graph n-1 where n is the number of nodes in G.

For multigraphs or graphs with self loops the maximum degree might be higher than n-1 and values of degree centrality greater than 1 are possible.

The centrality eigenvector module¶

class hcga.Operations.centrality_eigenvector.EigenCentrality(G)[source]¶

Centrality eigenvector

feature_extraction(args)[source]¶

Compute the eigenvector centrality for the graph G.

Eigenvector centrality computes the centrality for a node based on the centrality of its neighbors. The eigenvector centrality for node $i$ is the $i$-th element of the vector $x$ defined by the equation

\[Ax = \lambda x\]

where $A$ is the adjacency matrix of the graph G with eigenvalue $lambda$. By virtue of the Perron–Frobenius theorem, there is a unique solution $x$, all of whose entries are positive, if $lambda$ is the largest eigenvalue of the adjacency matrix $A$ ([2]).

G : graph
A networkx graph
args: list
Parameters for calculating feature_list
arg[0]: integer
number of bins
feature_list :list
List of features related to eigenvector centrality.
Implemented using networkx:
https://networkx.github.io/documentation/stable/_modules/networkx/algorithms/centrality/eigenvector.html#eigenvector_centrality

The measure was introduced by [1] and is discussed in [2].

[1]Phillip Bonacich. “Power and Centrality: A Family of Measures.” American Journal of Sociology 92(5):1170–1182, 1986 <http://www.leonidzhukov.net/hse/2014/socialnetworks/papers/Bonacich-Centrality.pdf>
[2](1, 2) Mark E. J. Newman. Networks: An Introduction. Oxford University Press, USA, 2010, pp. 169.
