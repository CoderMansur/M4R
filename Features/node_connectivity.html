The node connectivity module¶

Node connectivity class.

class hcga.features.node_connectivity.NodeConnectivity(graph=None)[source]¶

Node connectivity class.

compute_features()[source]¶

Compute node connectivity measures.

Node connectivity is equal to the minimum number of nodes that

must be removed to disconnect G or render it trivial. If source and target nodes are provided, this function returns the local node connectivity: the minimum number of nodes that must be removed to break all paths from source to target in G.

Implementation of networkx code:

Networkx_node_connectivity

This is a flow based implementation of node connectivity. The algorithm works by solving $O((n-delta-1+delta(delta-1)/2))$ maximum flow problems on an auxiliary digraph. Where $delta$ is the minimum degree of G. For details about the auxiliary digraph and the computation of local node connectivity see local_node_connectivity(). This implementation is based on algorithm 11 in 1.

1

White, Douglas R., and Mark Newman. 2001 A Fast Algorithm for

Node-Independent Paths. Santa Fe Institute Working Paper #01-07-035 http://eclectic.ss.uci.edu/~drwhite/working.pdf
