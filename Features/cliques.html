class hcga.Operations.cliques.Cliques(G)[source]¶

Cliques class

feature_extraction()[source]¶

Compute the clique number for the graph and the number of maximal cliques

A clique is the maximal subset of nodes such that all the nodes in this set are connected to each other. The clique number is the size of the largest clique in a graph.
