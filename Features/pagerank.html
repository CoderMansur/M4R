The PageRank module¶

class hcga.Operations.pagerank.PageRank(G)[source]¶

Page rank class

feature_extraction()[source]¶

Compute the PageRank of a network.

The PageRank ranks the nodes in a network based on the number of incoming links to that node. in undirected graphs, each edge is as an edge going in both directions.

Ggraph

A networkx graph

feature_list :list

List of features related to PageRank.

Implementation of networkx code:

Networkx_pagerank

PageRank computes a ranking of the nodes in the graph G based on the structure of the incoming links. It was originally designed as an algorithm to rank web pages.
