The eccentricity module¶

class hcga.Operations.eccentricity.Eccentricity(G)[source]¶

Eccentricity class

feature_extraction()[source]¶

Compute eccentricity for each node.

The eccentricity of a node is the maximum distance of the node from any other node in the graph 1.

Ggraph

A networkx graph
