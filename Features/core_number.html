The core_number module¶

class hcga.Operations.core_number.CoreNumber(G)[source]¶

Core number class

feature_extraction()[source]¶

Compute core number for each node

A k-core is the maximal subgraph that contains nodes of degree k or higher. The core number for a node is the largest value of k such that a k-core containing that node exists.
